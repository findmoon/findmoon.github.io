css hack

2016-05-04
  • 随笔
    • hack
  • hack

由于不同厂商的浏览器或浏览器的不同版本（如IE6-IE11,Firefox/Safari/Opera/Chrome等）对CSS的支持、解析不一样，导致在不同浏览器的环境中呈现出不一致的页面展现效果。这时，我们为了获得统一的页面效果，就需要针对不同的浏览器或不同版本写特定的CSS样式，我们把这个针对不同的浏览器/不同版本写相应的CSS code的过程，叫做CSS hack；
