git操作新建库

2016-04-27
技术 / git

桌面右键打开gitbash，命令行输入cd /c，转到C盘操作。

  • cd 切换文件夹时直接写文件夹名字即可，去掉/。如何从某以文件夹转到上一级目录？cd.. :切换到上级目录
  • rm删除文件或者文件夹 rm a.txt、rm -rf demo。删除文件夹命令必须加上 -rf
  • touch用于创建空文件。touch README.md 创建README.md的文件，内容为空
  • mkdir用于创建文件夹，mkdir demo创建名叫demo的文件夹
  • cp复制文件或者文件夹
    cp a.txt b.txt：把a.txt拷贝一份放到当前目录，起名叫b.txt
    cp -R demo demo2：把demo文件夹复制一份，起名叫demo2

ls：查看c盘下文件/内容

mkdir :新建文件夹。（touch：新建文件）

pwd :查看当前操作的文件的路径

git clone ssh：将git-master内容克隆到本地,yes连接

如果出现，permission denied 以及fatal（错误），需要创建公私钥连接

  • 创建公私钥
    ssh-keygen -t rsa -b 4096 -C “your_email@example.com”
    （Creates a new ssh key, using the provided email as a label
    Generating public/private rsa key pair.
  • 查看sshkey相关说明

git statue，查看状态，出了什么问题
pull之后，git默认把远程库和本地做个合并，合并后就需要生成新的提交，必须输入一些文字告诉做了什么事情，默认打开vim编辑器（模拟的linux环境），输入 有两种模式，命令模式和编辑模式，编辑模式才能输入一些文字。输入i（insert）就可以编辑输入文字，ESC退出insert，进入功能模式。然后输入：w（writer）q（quit）【保存退出】，相当执行了commit

clone完后，cd 到文件夹下操作

git bash中使用Ctrl+上下键借用以前的命令

（cd project/[project为文件夹]）
