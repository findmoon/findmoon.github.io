gitbash命令

2016-04-27
技术 / git

不管是windows下的git-bash 还是 Mac下的终端，内核都是linux环境，我们可以使用linux的命令去操作。
下面介绍几个常见的Linux命令：

  1. ls
    ls用于查看当前文件夹下有哪些文件，用法有
    ls
    ls -al
  2. mkdir
    mkdir用于创建文件夹，如下创建一个名叫demo的文件夹
    mkdir demo
  3. cd
    cd用于切换目录
    cd other
    切换到当前文件夹下的other目录
    cd ..
    切换到上级目录
  4. rm
    rm删除文件或者文件夹
    rm a.txt
    删除a.txt
    rm -rf demo
    删除demo文件夹
  5. cp
    cp复制文件或者文件夹
    cp a.txt b.txt
    把a.txt拷贝一份放到当前目录，起名叫b.txt
    cp -R demo demo2
    把demo文件夹复制一份，起名叫demo2
  6. touch
    touch用于创建空文件
    touch README.md
    创建README.md的文件，内容为空
    vi
    vi命令用于吊起linux默认的vi编辑器进行编辑。
    vi a.txt
    用vi编辑器编辑a.txt文件
    vi的使用参考这里http://www.runoob.com/linux/linux-vim.html
